# 前言

昨天晚上感觉对古人的 “吾日三省吾身” 有了更深的感触。

不知道大家有过这样的经历没？就是到了晚上，特别是过了凌晨 12 点之后，感觉自己就非常清醒，特别能正视问题，非常能接受别人的指正，批评等等

# 为什么古人反省的时间要选在晚上

一是：

零点，昨天结束了，今天还没开始。

这时候，世界安静了，自己也安静了。没有电话，没有微信，没有那些让你分心的琐事。

这个时间点 —— 不属于任何一天。在时间的缝隙里，人最容易看清自己。人在这一时刻，不再像是生活参与者，更像是旁观者。“当局者迷，旁观者清”。

二是：

到了晚上，人已是精疲力尽。

# 打架的系统一与系统二

相信你听过那个大脑的 “两套系统”。系统一，依靠思考，缓慢，低效，还不可靠；系统二，依靠记忆，快速，高效，还可靠。

那如何高效利用自己的系统二呢？

在《身心合一的奇迹力量》一书中有这样一段话：

绝大部分运动领域的运动员都用相似的表达，最好的运动员都知道如果他们” 想” 做出顶级表现，这种表现绝对不会来。

当然，无意识地打球并不是说打球的时候一点没有意识。那将是相当的困难或者说是不可能的。

事实上，当某人打 “疯” 了的时候，事实上对球和球场以及在必要的情况下对他的对手都保持着专注层次更高的 “意识”（此 “意识” 做作动词），只是他并不有意识地给他自己很多指令（来确保击出想要的球），比如如何来击这次球，如何改正过去的错误或如何来重复刚才那一次漂亮的击球过程。他是有意识的，但没有思考，也没有过度 “努力”。

处于这种状态的运动员知道他想要球去到什么位置，但他又不是在非常努力地将球打到那里去。它就是发生了，并且精准度还经常性地比他希望的还要好。这时运动员似乎完全沉浸在这种连续的流畅的击球过程中，能量进一步因协调得以聚集并爆发，导致球的速度和精确度到一个更高的层次。这种好的击球状态会一直持续，直到他的意识开始介入，并 “想”（欲望开始产生）要将这种状态保持下去；（他开始思考和分析刚才的击球过程，以便自己可重复性使用，）但结果是一旦他尝试想留住那种感觉，他马上就找不到了，整个一个” 来无影，去无踪”（如果开始着急，去得更远）。

大脑一次只能运行一种系统。所以想让系统二上线，就首先想想如何让系统一下线。让大脑短时间接收海量信息，导致系统一过载，这时系统二自然上线。

回归正题，正因为大脑极度疲惫，所以晚上主要是系统二在工作，这时从大脑的系统角度来说，大脑基本上只在使用记忆检索引擎工作，几乎没在思考，自然没有杂七杂八的想法内耗自己。

从资源思维来说，正因为很疲惫，所以注意力资源紧缺，甚至由于过于紧缺，直接将原本用来 “思考别人对这件事的看法”，“我在做事有没有失误” 等等思考的资源抢了过去。

# 这种力量怎么运用在生活中呢？

# 创业上：

穷人的孩子早当家。“中原闲人创业记” 中也说过 “当一个人手里没有多少资源的时候，潜力往往就会爆发，潜力爆发了，资源往往就并不那么重要的，但不那么重要不代表没有资源就可以，资源必须有，但已经不需要太多”

特别是非洲人民的创造力往往正是穷才具有这么强的创造力；很多巨著的作者往往是生活条件很不好的人……

# 自我复盘上

面对批评时，重点是如何改进？
面对失败的经历时，重点是怎么吸取教训？
面对预料之外的诱惑时，重点是怎么坚定初心

# 信息处理上

一直放在收藏夹吃灰的文章，是高质量干货，还是伪装的营销水货？

堆积如山的图片，想找自己想要的图时，翻了半天都翻不到，是不是垃圾无用图片太多了？

一群加完 1 年都没说过一句话的躺尸好友，朋友圈一看，都不是自己想看闺蜜的朋友圈，还得费劲的往下划？

# 扩展 -- 酒局文化

以前中国还没有那么富裕的时候，有人说过，这酒店开的比买牛奶的店还多，牛奶都喝不起，还开这么多酒店……

然后为什么酒局文化这么兴盛？中国人很会含蓄和委婉，想短时间看清一个人很困难。

酒局文化正是解决这个问题的关键，当一个人喝醉了之后，资源不足，平常那种伪装就，很难维持。这是很容易看出一个人真正的样子。
平常斯斯文文彬彬有礼的人喝完酒后突然大吼大叫； 平常无比外向豪迈的大汉，喝完酒后沉默不语……

然后敬酒又是酒局文化中另一个延伸，给了一个量化标准。
什么量化标准呢？现在的人都知道喝酒伤身。但通常喜欢敬酒，就是你接了这杯酒，就是人家的一个面子，间接表明对待人家的态度是怎么样？
即便喝酒伤身，咱们俩还是好朋友，所以说还是要喝一杯。

除了酒局文化的扩展以外，还有那个 “快乐的时光总是短暂的”。正因为快乐，所以说忍不住想多投点，就没有资源分配到感知时间上了。而水课和无意识的会议则是相反，注意力多了没地方放，越注意时间，就感觉时间过得越慢。
