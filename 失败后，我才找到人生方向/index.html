# 失败后，我才找到人生方向

# 引言

上一讲主要分享了决策的四大支持。今天我想分享一件我觉得我大一干的挺有意义的事：我花时间收集了学校近三年所有省部级一类以上的比赛、项目申报、实验室分布以及历年毕业生考证情况的信息，结合招聘公司的需求，反推出了一份从容不迫的预设简历和清晰的技能升级路径。信息收集一直是我的强项，但这个过程并不总是一帆风顺。我踩过坑，也调整过思路，今天就聊聊我的真实经历和教训，希望能给你一些启发。

# 别被 “高大上” 迷惑

在收集信息的时候，别老想着要找那些 “别人觉得很厉害、很羡慕” 的东西。就像参考里说的，别为了 “帮助他人” 而牺牲自己的兴趣。你的职业规划是你自己的，兴趣为王！只要你对某件事有兴趣，就一定能找到志同道合的人，一起把兴趣变成事业。别让外界的 “高大上” 蒙蔽了你的双眼，真正的宝藏往往藏在你内心最真实的声音里。

想想看，如果你一心只想做 “别人眼中的成功者”，最后可能会迷失自己。你的职业规划不是给别人看的，而是为了让自己过得充实、有意义。所以，别管别人怎么想，跟着自己的兴趣走，路会越走越宽。

刚开始做职业规划时，我总觉得要追那些 “高大上” 的东西，比如热门行业或者别人眼中的成功。我一度盯着金融、互联网这些 “厉害” 的领域，想着 “得干点让别人羡慕的事”。但我很快发现，这种想法让我很累，因为我根本不感兴趣。

有一次，我接触到了 ChatGPT，发现自己对 AI 辅助个人思考特别好奇，尤其是探索哪些说不出来的感觉和分析生活中的各种各样的问题。这让我意识到，兴趣才是王道。与其追别人眼中的 “高大上”，不如听听自己内心的声音。

之前，我为了 “看起来厉害”，参与了一个车队比赛，结果完全没动力，准备到一半就放弃了，浪费了时间还打击了自信。后来我明白，职业规划不是给别人看的，而是让自己过得充实。所以，别被外界蒙蔽，跟着兴趣走，路才会越走越宽。

# 决策靠的是 “关键数据” 而不是 “海量数据”

随着信息技术的进步，数据量急剧增加。也就是数据大爆炸的时代。人们开始依赖大数据来做决策，但发现海量数据并不总能提高决策质量。 而现在越来越多的研究和实践表明，关键数据在决策中的作用更为重要。

道理很容易懂，比如说：
在人这个生态系统中，平衡很重要。重要到什么程度。
以前长期在海上航行的人经常会得坏血病，仅仅是缺少了微量的维生素 C，如果知道了就好办准备一些橘子即可。但是如果不知道，坏血病将会是这艘船的死神，疯狂收割着每一位船员的生命，就算你有再好的医生，再多的大补之物，也救不了你。

我也是这样认为的，并将其当做我做职业规划时的指导思想。
我职业规划总结起来，可概括成：预设简历 + 超越简历作品集。

而我认为做不出预测简历的核心原因在于缺少关键数据，而不是需要海量数据。只要掌握了关键信息，例如往年学生或学姐在简历中填写的内容，制作一份预测简历就会变得轻而易举。关键数据在手，很多问题都能迎刃而解。比如，想知道哪些老师更厉害，只需查看往年比赛获奖情况数据表中老师名字出现的频率，就能快速得出结论。

此外，“预制简历” 并不是随意 “瞎填” 本专业的奖项来包装自己，而是知道本学校本专业的所有比赛获奖情况，通过对每个比赛进行含金量与易得性的权衡，列出一份清单，然后再通过获奖情况数据表中老师或学长的名字，去联系进一步了解比赛详情，最后再筛选一遍。然后提前为比赛做准备。这里有个更简单易懂的逻辑：就是既然我们学校能在某个比赛能拿到银奖，那凭什么我提前知道这个比赛，并且提前准备几个月的人会比不上哪种临时加入比赛获奖的人呢。

然后，预设简历最大的好处，虽然我们都懂，但是为什么生活依然这么差？因为脑子懂不代表身体懂了，况且脑袋是不是真懂还是两说呢！

为什么说脑子懂不代表身体懂了？

你读了书、看了视频，懂了很多道理，比如 “时间管理能提升效率”。 没有行动 - 但你还是习惯性拖延，没把时间管理用起来，大脑的 “懂” 停留在表面。生活没变 - 结果呢？道理堆满脑子，生活却还是老样子，甚至更糟，因为你还多了份 “知道却做不到” 的愧疚感。

再举个通俗易懂的例子：
想象你学骑自行车：

  • 大脑懂了：知道要蹬踏板、保持平衡。
  • 身体不懂：第一次骑还是摔得鼻青脸肿。
    只有多摔几次，身体才会 “懂”，骑车变成自然反应。生活中的道理也是如此，光靠脑子记不住，得靠身体去 “练”。

再说一个非常重要的认知：方法本身只是理论的一个使用说明书，并没什么大不了的。真正重要的是对理论的理解。要不然你想想最开始的方法是从哪里来的。

# 学校：你的信息宝库

学校信息多得像个宝库，但怎么挖是个技术活。下面是我的五个渠道，每个都踩过坑，也总结了经验。

# 1. 社交媒体和论坛：从乱淘到精选

社交媒体和论坛（比如贴吧、小红书、学校 QQ 群）就像个大集市，学长学姐常分享经验。但刚开始，我被各种帖子带着跑，看了一堆吐槽和炫耀，啥也没学到。

转变：后来我学会筛选，只看有干货的内容，比如职业路径或比赛经验。一次，我在小红书看到学姐分享如何准备省赛，照着做了，效率翻倍。

怎么筛选？ 你可以关注那些有干货的帖子，比如学长学姐的职业发展路径、他们遇到的挑战和解决方法。别浪费时间在无意义的吐槽或炫耀上。你的时间宝贵，要用在刀刃上。

# 2. 官方校园网站：从表面到深度

学校的官网上有很多官方信息，比如专业设置、课程安排等。这些信息很靠谱，是你了解学校的好起点。但别忘了，官网上的信息可能比较正式，你还需要从其他渠道获取更多细节。想想看，官网是学校的 “门面”，但真正的生活气息，还得从其他地方找。

比如，你可以看看学校的本科生培养方案、往年比赛获取情况和实验室情况，这些数据可以对你想要规划绩点，考研和就业都有很大帮助。

# 3. 学生会和校团委：从数字到故事

学生会和校团委有各种数据，比如四级通过率、毕业生去向等。这些数据能让你了解学校的整体情况。但别只看数字，要试着去了解背后的故事，比如为什么某个专业的四级能有 60%，这个 60% 是怎么算的？四级考生除了大一的，还二战，三战的嘛？数量多不多？总之数据是死的，人是活的，多问问 “为什么”，你才能发现更多。

举个例子，如果你想了解往年毕业生考过的证书情况，可以通过你在那些组织的朋友，去查的。在不泄露学校隐私的情况下，让你朋友在做这部分数据统计工作的时候，顺便看看数据不过分的。完了回头请别人吃顿饭即可。

# 4. 导员：从不敢问到真诚聊

在大学中，你一年都不一定见到班主任一面。但你能经常见到导员，导员基本上什么事都管，他们既有经验，也有权限。如果你真心想了解某件事，不妨去请教他们。但记住，真诚是关键，别装模作样。导员也是人，他们更愿意帮助那些真心求知的学生。拿出你的诚意，他们会给你惊喜。

怎么和导员交流？ 别一上来就问一大堆问题，先聊聊近况，表达你的困惑和需求。然后，认真听他们的建议，别打断。导员的时间也很宝贵，尊重他们的时间，他们会更愿意帮你。

# 5. 暑期评建办志愿者：从被动等到主动出击

学校的教学评估资料里藏着很多有用的信息。
如果你有机会成为暑期评建办的志愿者，就能接触到这些宝藏。
主动出击，你会发现很多惊喜。机会总是留给有准备的人，别等机会来敲门，自己去敲机会的门。
特别值得注意的一点是，第五点才是我 “一战而定” 的关键信息收集的主要渠道，也是我的独门绝技，这也是其他网上文章质量为什么比不上这篇文章的关键区别。在每年暑假前，去打听一下暑期评建办有没有招志愿者。一般每个学校每年都有，因为每年必有专家来教学质量评估。

为什么说这点绝对关键？

从转化率的角度分析：
我们都知道生态系统的食物链下一级到上一级的效率大约在 12% 左右（具体记不太清楚了哈）。在自媒体尤为显著，多一步就是 10% 的转化率，1000 播放量。

# 学长学姐：如何正确的 “抄作业”？

学长学姐走过你将要走的路，他们的经验是你的宝贵财富。别害羞，多和他们交流，了解他们的故事，从中学习。但记住，每个人的路都不一样，别盲目模仿，要找到适合自己的方式。学长学姐是榜样，但人生的主角是你自己。

怎么和学长学姐交流？ 你可以通过社团、活动或者社交媒体联系他们。准备几个具体的问题，比如 “您在大学期间遇到的最大挑战是什么？” 或者 “您是怎么选择职业方向的？” 别问太宽泛的问题，具体的问题能让你得到更具体的答案。

# 链接学长学姐的技巧：Q&A 指南

说到学长学姐，链接他们也是一门技术活，尤其面对警惕性高的 “硬茬”，比如有对象的学姐。我和朋友聊过这事，总结了一套简单实用的方法，整合成 Q&A 指南，分享给你。

  • Q1：如果一个学弟突然加你微信，你会咋想？
    A1：学姐可能想 “谁啊？啥目的？”，尤其女生警惕性高，第一反应是防着点。所以，开场白得稳，别让人觉得你有啥 “企图”。

  • Q2：学长学姐被加时会咋想？通过率高吗？
    A2：一般不轻易通过，除非你自我介绍靠谱。我加过 1000 多人，通过率 30%，300 个左右。警惕性高的，比如有对象的学姐，可能更难，得有策略。

  • Q3：咋找到学长学姐的联系方式？
    A3：QQ 群、官网是好渠道。我常模糊说 “官网上看见的学姐学长都加了一个遍”，显得自然不刻意，别让人觉得你专门盯着她。

  • Q4：加学长学姐的第一句话咋说？
    A4：简单直接，比如 “你是山科学弟，就是希望跟学姐学习学习”。我试过问 “你今天看见教官了吧？” 或者 “我在参加职业规划大赛，想问问学姐对专业的看法”，谦逊又不突兀。

  • Q5：对方怀疑咋办？比如问 “你咋知道微信的”？
    A5：模糊回答 “官网上看见的学姐学长都加了一个遍”，显得你不是针对她。我还调侃过 “天天蹭校外资源”，轻松化解怀疑。

  • Q6：咋提高通过率和信任感？
    A6：用 “学习学习” 拉好感，强调 “你是山科学弟”，低姿态 + 广撒网，别让人觉得你有私心。红包别送，太功利。

  • Q7：链接学长学姐有啥策略？
    A7：广撒网（1000 人加 300 个）+ 精准打击（针对警惕性高的用 “官网都加遍”），结合用效果好。节奏要稳，别太急。

  • Q8：聊不下去咋办？
    A8：对方冷淡就撤，别硬聊。比如问 “学姐对教官有啥建议吗？” 她回 “有” 就不接，那就停，别尬聊。

  • Q9：有啥要注意的坑？
    A9：别让人误会你有私心，全程学术味儿。警惕性高的学姐有对象，别太随便，别乱传联系方式，小心小人。

  • Q10：为啥学长学姐警惕或不通过？
    A10：怕骗子、怕麻烦，有对象的更怕误会。用 “学习学习” 这种无害理由破冰。

  • Q11：咋判断对方对你的期待？
    A11：看回话热情。冷淡可能没期待，愿意聊可能觉得你有用。试探问 “学姐对专业有啥建议吗？” 看反应。

  • Q12：对方冷淡或防备的表层和深层原因是啥？咋应对？
    A12：表层是不熟，深层是怕麻烦或误会。用 “山科学弟”“官网都加遍” 破冰，显得真诚无害。

  • Q13：链接学长学姐的经验教训是啥？
    A13：开场稳（“希望学习”）、理由无害（“官网都加”）、边界守（学术味儿）、心态好（细水长流），慢慢就熟了。

# 信息收集的技巧

收集信息不是瞎逛，要有技巧。以下是几个小贴士，帮你事半功倍：

  • 知道自己要啥：先想清楚你想了解什么，再去收集信息。别像无头苍蝇一样乱撞。
  • 多渠道获取：别只依赖一个渠道，多方打听。信息越全面，你的地图越清晰。
  • 别轻信：对信息要有点怀疑精神，别啥都信。学会分辨真假，别被忽悠。
  • 记下来：把有用的信息记下来，方便以后用。别指望脑子能记住所有东西。
  • 持续关注：信息会变，要时不时更新你的信息库。别让你的地图过时。

举个例子，如果你对某个行业感兴趣，你可以关注行业大咖的微博、公众号，参加相关的线上线下活动，甚至尝试联系业内人士请教。别怕麻烦，主动出击，你会收获更多。

# 结语：听从内心的声音

信息质量排序：暑期评建办 > 导员 > 官网 > 社交媒体。我的经验是，听从内心，别被 “高大上” 牵着走。兴趣是你最大的动力，失败是最好的老师。我踩过坑，比如收集无用信息、盲目模仿，但每一次调整都让我更靠近目标。职业规划是持续的过程，别急，慢慢来，你会找到自己的路。
