# 为什么会为了 “找不到想做的事”，而一直彷徨呢？

在上一篇文章中，我们揭穿了 “职业规划无用” 等迷思，强调了规划的重要性。然而，许多人在职业选择时仍感到迷茫：为什么收集到的信息总是不够全面？如何做出不后悔的决策？我认为，答案在于四大支持：

四大支持核心含义
全面的信息掌握行业、公司、职位等外部信息
科学的决策流程用结构化方法评估选项，减少盲目
深入了解自己明确自己的兴趣、优势和价值观
自己拿决定基于自我认知独立判断，不盲从他人

本篇将聚焦于 “全面信息中的了解自己” 和 “建立系统” 的方法。至于 “科学的决策流程” 和 “自己拿决定”，我们将在后续文章中深入探讨。准备好了吗？让我们一起踏上这场自我探索与成长的旅程吧！

# 全面信息中的 “了解自己”

在职业规划中，全面的信息是成功的关键。《孙子兵法》有云：“知己知彼，百战不殆。” 这句话不仅适用于战场，也同样适用于我们的职业发展。然而，在现实中，问题往往不在于 “不知彼”，而在于 “不知己”。我们习惯于搜集外部信息 —— 行业趋势、公司文化、职位要求，却常常忽略了对自己的深入挖掘。全面的信息不仅包括外部的 “彼”，还包括内部的 “己”，具体可以分为以下四个来源：

自己学校
学长学姐企业

如果我们只关注外部信息，而对自己的兴趣、能力和价值观一知半解，就很难做出真正适合自己的选择。这就像在地图上找路，却不知道自己身在何处 —— 方向再明确，也只是盲目的奔跑。

# 为什么 “了解自己” 如此重要？

在职业规划中，了解自己（知己）是至关重要的第一步。很多人觉得自己天生就了解自己，但事实往往并非如此。知己不仅意味着知道自己喜欢什么、擅长什么，还包括认识自己的局限、情绪模式以及深层的职业追求。如果没有这种自我认知，职业规划就如同无根之木，风一吹就倒。

就像我大一去找实习一样，想着去寻找嵌入式的实习，其它方面准备的十分充足，但是当别人进行追问我具体要找什么样的实习时，我才发现我对我自己实际上要找什么样的还是感到有些迷茫。我要找嵌入式实习有些模糊，到底是嵌入式开发，还是嵌入式测试呢？我想不清楚。表面上看，我的目标很明确，但实际上，我对自己真正的需求一片迷雾。一开始方向就偏了，后面只会越走越远。这让我意识到，外部信息固然重要，但如果 “知己” 这一步没做好，再多的准备也只是空中楼阁。

这种迷茫并不少见。很多人会在职业选择的十字路口徘徊，因为他们不清楚自己到底想要什么。是因为薪资高而选一份工作，还是因为兴趣而坚持一个方向？没有自我认知的支撑，这些选择往往经不起时间的考验。

# 如何深入了解自己？

要真正了解自己，光靠直觉是不够的。我尝试过一些工具和方法，逐渐摸索出了自己的轮廓。以下是几个我认为行之有效的途径：

# 1. MBTI：性格的初步画像

MBTI（Myers-Briggs Type Indicator）是一个广为人知的性格测试工具，通过 16 种性格类型，帮助我们了解自己的倾向。比如，我是内向（I）还是外向（E）？是靠直觉（N）还是观察（S）？测试结果告诉我，我是 INTP 类型 —— 喜欢独立思考、擅长分析问题。这让我意识到，我可能更适合需要深度钻研的工作，而不是频繁社交的岗位。不过，MBTI 也有局限，它过于抽象，无法直接告诉我该选什么职业，只能作为起点。

# 2. 《发现你的天职》：挖掘潜意识的 “习性”

相比 MBTI，我更推荐《发现你的天职》这本书。它让我重新定义了 “擅长的事”。书中提出了一个公式：
真心想做的事 = 重视的事 × 擅长的事 × 喜欢的事。
其中，擅长的事不是你刻意练出来的技能，而是能创造成果的潜意识、情绪与行为模式，也就是你的 “习性”。

“习性” 是什么？它是你自然而然流露出来的行为。比如：

  • 你是不是总在人群中默默观察别人的反应？
  • 你是不是一有灵感就忍不住动手尝试？
  • 你是不是总在思考如何让事情更高效？

这些习惯往往是你潜意识里的优势，但因为太自然，反而容易被忽略。我曾以为自己只是 “喜欢分析”，但通过反思，我发现这是一种根深蒂固的习性 —— 无论是在学习、工作还是生活中，我总会不自觉地拆解问题、寻找规律。这种 “反射性动作” 让我很少主动提醒自己 “我在分析”，所以需要刻意回顾，才能发现这些隐藏的 “擅长的事”。

# 3. 反思过往决策：从经历中找线索

除了书本和测试，我还翻看了自己当年的志愿填报存档。那时候选专业，我考虑了兴趣（技术）、能力（理科成绩不错）和未来发展（嵌入式领域有前景）。回顾这个过程，我发现自己对 “动手解决问题” 有种天然的偏好。这跟我的 “分析” 习性一脉相承，进一步印证了我的优势。

# 4. 更多工具：拓宽自我认知的边界

后来，我还尝试了其他方法，比如霍兰德职业兴趣测试，它根据兴趣类型（现实型、研究型、艺术型等）推荐匹配的职业领域。还有爱好清单，根据 mbti 列出可能的爱好并且实践，感知自己究竟喜欢什么。另外还有原生家庭报告，追果溯因找到一些性格的来源等等。这些工具像一面面镜子，帮我从不同角度看清自己。

通过这些方法，我逐渐明白，了解自己不是一蹴而就的，而是一个需要耐心挖掘的过程。就像挖矿，你得一层一层剥开表面的土，才能找到真正的金子。

# 建立系统：持续探索与成长

# 从 “了解自己” 到 “持续成长”

了解自己是第一步，但光停留在 “知道” 是不够的。如何让这种认知不断深化，变成职业规划的动力？答案是建立系统。系统思维能让你像科学家一样，持续收集信息、分析反馈、调整方向，把职业规划从静态的目标变成动态的旅程。

# 目标导向 vs. 系统导向

在知识管理领域，有一个很火的概念：建立系统而不是设立目标。为什么强调 “系统” 而不是 “目标”？这两者到底有什么区别？让我们用一个大篇幅，把这个概念讲透。

# 目标导向：短跑的思维

目标导向是一种短期驱动、结果导向、单点突破的思维方式。比如，我说：“我要一个月学会编程。” 哪怕目标定为 “两年学会”，它依然是短期的，因为一旦达成（或失败），你就停下来了。你所有的努力都围绕这个结果，拆分成里程碑，按部就班往前冲。这种方式简单直接，但也有局限 —— 它只关注一个点，忽略了整体。

# 系统导向：马拉松的智慧

系统导向则完全不同。它不追求单一的结果，而是搭建一个框架，让你每天都在其中运转。比如，你可能没有明确说 “我要用这系统干什么”，但因为你每天都在用它处理信息，它会自然进化。你会发现问题（“这部分不好用”），调整策略（“改改试试”），最终带来整体性的提升。这种方式更像种树，慢慢长大，而不是盖房子，盖完就完事。

为了讲清楚两者的区别，我们用一个生活化的例子 —— 坚持早起，来看看它们如何运作。

# 目标导向下的 “坚持早起”

假设我定了个目标：每天早上 6 点起床。为了实现它，我会列计划：晚上 11 点睡觉，6 点设闹钟。每天醒来后，我只关心一件事 ——“我起床了吗？” 如果连续一个月成功，我就觉得自己 “完成了”，任务结束。如果失败了，我就可能放弃，觉得自己 “不适合早起”。

这种方式下，我收集的信息很窄：睡多久合适？用什么闹钟好？一旦目标达成或失败，我就不再深入思考。比如，我不会去想为什么某天精神特别差，或者周末为什么总起不来。因为我的焦点只在 “6 点” 这个点上，其他因素都被忽略了。这样一来，我对早起的理解很片面，经验也很难迁移到别处，比如改善睡眠质量。

# 系统导向下的 “坚持早起”

如果用系统导向，我会换个玩法。我会建一个早起系统：每天记录睡眠时间、起床时间、精神状态（1-5 分）、睡前活动（刷手机还是看书）、饮食情况等。每周花 10 分钟回顾，分析哪些因素影响了早起。

比如，我发现：睡前玩手机半小时，入睡拖到凌晨，第二天像踩棉花；晚餐太油腻，早上头昏脑涨；周末作息乱了，起床就废了；压力大时反而早起，因为精神紧绷。根据这些反馈，我调整策略：睡前看书，晚餐清淡点，提前到 7 点吃。下一周再看，入睡早了 20 分钟，精神从 3 分提到 4 分，效果不错。

这种方式下，我不仅关心 “能不能早起”，还探究 “为什么能早起” 和 “为什么起不来”。我收集的信息更全面：睡前习惯、饮食、心理状态，甚至周末和压力日的差别。调整后，这个系统不仅帮我早起，还能优化生活其他方面，比如饮食习惯或工作效率。

# 两者的本质区别

简单来说，目标导向像短跑，盯着终点冲，跑完就停，信息收集和经验积累都有限。系统导向像马拉松，边跑边调整，不仅能到达终点，还能跑得更远。它通过持续的反馈和优化，让你对问题有更深的理解，方法也能迁移到其他领域。

# 如何在职业规划中建立系统？

明白了系统思维的好处，我们该如何把它用在职业规划上？以下是一些具体建议：

# 1. 建立反馈循环

职业规划不是定个目标就完事，而是要像科学家做实验：行动 → 记录 → 分析 → 调整。比如，你想学编程，每天记录学习时长、难点、解决办法，每周回顾一次，看看哪些方法有效，哪些需要改进。这样，你不仅能学会编程，还能掌握学习新技能的规律。

# 2. 打造个人情报网络

就像文章开头说的，全面的信息很重要。除了自己，你还可以利用学长学姐、学校资源和企业信息。比如，我会定期跟学长聊聊他们的职业经历，参加学校的职业讲座，甚至主动联系企业 HR 了解岗位需求。这些信息就像拼图，帮我拼出更完整的职业图景。

# 3. 记录与反思的习惯

我喜欢用笔记本或 Notion 记录每天的思考。比如，今天面试了什么岗位，感觉如何？最近学的技术有没有用？每隔一个月，我会翻看这些记录，问自己：我离目标近了吗？ 我还需要什么？ 通过这种方式，我能看到自己的成长轨迹，也能及时调整方向。

# 4. 从小系统开始

如果你觉得 “系统” 听起来很复杂，可以从小处入手。比如，建一个 “每周复盘系统”：每周五花 15 分钟，写下这周的收获、问题和下周计划。坚持几周后，你会发现自己的思路越来越清晰。

# 更深入的探讨：自我认知与系统的融合

# 自我认知的深化：从 “知道” 到 “内化”

了解自己不是终点，而是起点。比如，我知道自己擅长分析，但这只是表面的 “知道”。要真正内化，我需要问：这种擅长在哪些场景最有用？ 它会让我在哪些地方吃亏？ 通过系统的记录和反思，我发现分析能力让我在解决问题时得心应手，但在需要快速决策时容易犹豫。这种洞察让我更有针对性地调整自己。

# 系统的力量：从碎片到整体

职业规划中的信息往往是碎片化的：今天听了个讲座，明天看了篇行业报告。系统思维就像一根线，把这些碎片串起来。比如，我会把讲座笔记、行业趋势和个人反思都记在 Notion 里，定期整理，找出它们之间的联系。慢慢地，这些碎片就变成了一个完整的知识网络，帮我看清职业的全貌。

# 克服迷茫：系统的动态调整

迷茫往往来自信息的不完整。通过系统，我可以主动填补空白。比如，我发现自己对嵌入式测试不感兴趣，就调整方向，深入研究开发岗的需求。这种动态调整让我在迷雾中找到路，而不是一味彷徨。

# 结语：从彷徨到清晰的旅程

职业规划中，为什么会收集不到全面的信息？ 因为我们往往只看外在的 “彼”，忘了内在的 “己”。了解自己是起点，建立系统是保障。两者结合，你才能从被动的收集者变成主动的探索者。

这不是一蹴而就的过程，而是一场需要耐心和反思的旅程。就像种一棵树，你得先挖土、埋种、浇水，然后看着它慢慢长大。希望这篇文章能给你一些启发，让你在职业道路上走得更稳、更远。
